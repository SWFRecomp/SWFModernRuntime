SWFModernRuntime

Flash SWF files running natively in WebAssembly

What is this?

This project brings Adobe Flash SWF files to modern web browsers through static recompilation to WebAssembly.

No Flash Player. No emulation. Just pure native code compiled from SWF bytecode.

How It Works

  1. SWFRecomp parses SWF bytecode and generates C code
  2. Emscripten compiles the C code to WebAssembly
  3. This runtime executes ActionScript and renders graphics in browser
  4. Result: Native performance with no Flash Player needed!
